# Certificate PDFs Guide

## PDF Certificates
Place your certificate PDF files in this folder.

### Already Configured:
- `world-bank-certificate.pdf` - World Bank Policy Makers certificate

### Instructions:
1. Export or download your certificates as PDF
2. Rename them clearly (e.g., `mckinsey-forward.pdf`, `bloomberg-bmc.pdf`)
3. Add the path to certificates.json:
   ```json
   "pdfPath": "/certificates/filename.pdf"
   ```

### File Naming Convention:
- Use lowercase
- Use hyphens instead of spaces
- Keep names short and descriptive
- Example: `cambridge-advance-certificate.pdf`

## Note
PDFs will be downloadable from the certificates page.
