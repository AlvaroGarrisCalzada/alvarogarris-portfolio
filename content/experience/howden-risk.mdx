## Overview

Supported risk assessment and modeling initiatives at Howden, a global insurance broker, applying quantitative methods to evaluate and price complex insurance risks.

## Key Responsibilities

- **Risk Analysis**: Analyzed loss data and built statistical models for risk quantification
- **Model Development**: Developed predictive models for claims frequency and severity
- **Portfolio Analysis**: Evaluated insurance portfolio risk profiles and concentration risks
- **Automation**: Created automated reporting tools for risk metrics and KPIs

## Key Projects

- **Claims Predictive Model**: Built machine learning model to predict claim likelihood with 85% accuracy
- **Portfolio Dashboard**: Developed interactive Power BI dashboard for real-time portfolio monitoring
- **Risk Concentration Analysis**: Identified portfolio concentration risks across industries and geographies

## Technical Skills Applied

- Statistical modeling using R and Python (scikit-learn, pandas)
- Database queries and data manipulation with SQL
- Monte Carlo simulation for scenario analysis
- Data visualization and reporting automation
